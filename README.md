# 行测复盘系统 · xingcefupan

> 把「做套卷」变成一套可复制的提分流程：**记录 → 诊断 → 行动 → 计划 → 打卡**。

这是一个基于 Streamlit 的行测（行政职业能力测验）学习与复盘小工具，主要用于记录每一套行测模考试卷的表现，自动拆解模块短板、时间黑洞，并给出可执行的训练建议和一周训练计划，帮助考生在省考、国考备考阶段更体系化地刷题和复盘。

---

## ✨ 项目功能概览

### 1. 账号系统 & 多人使用

- 支持**注册 / 登录**，内置一个 `admin` 管理员账号（可新增/删除用户）
- 每个账号的数据（成绩、复盘记录、打卡情况等）**独立存储**，适合个人使用或小规模教学生使用

### 2. 🏠 数字化看板

- 展示最近一套模考的核心指标：
  - 总分 / 正确率 / 总用时 / 得分效率 / 近 5 次均分
- 自动识别：
  - **当前短板模块**（正确率最低）
  - **时间黑洞模块**（超时最多）
- 可视化图表：
  - 能力雷达图：按模块查看强弱
  - 分数分布直方图：观察分数稳定性
- 若填写了复盘记录，还会显示：
  - 近 N 天（可配置）错因统计（不会 / 不熟 / 审题坑）
  - 模块错题 Top 排行

### 3. 📑 单卷详情（核心复盘页）

- 可选择任意一套历史试卷，查看：
  - 本卷总分、正确率、总用时、得分效率
- 系统自动生成：
  - **正确率最低 TOP3 模块**
  - **超时最多 TOP3 模块**
  - 每个模块对应一张「浅色提示卡」，包含：
    - 模块名称（例如：`数量关系`、`资料分析`、`言语-逻辑填空` 等）
    - 正确率、用时 vs 计划用时
    - 针对该模块的**具体行动建议**（例如：数量设置每题 60 秒上限、资料每篇 6 分钟上限、言语每天专项 20 题等）
- 自动生成下一天的训练建议：
  - 「✅ 明天怎么练（只给 3 条，能执行）」——直接照着做就行
- 还提供：
  - 各模块小卡片（政治/常识/言语/数量/判断/资料），一屏快速扫完
  - 一键导出本卷复盘摘要（Markdown），方便复制到笔记/备忘录

### 4. 🧠 复盘记录

- 按套卷 + 模块记录「错题结构」：
  - 错题数
  - 错因拆分：**不会 / 不熟 / 审题坑**
  - 一句话原因（如：基期现期看反、转折句没抓、速算失误）
  - 下次只改 1 个动作（如：资料每篇 6 分钟上限、数量每题 60 秒上限、填空每天 20 题）
- 系统会智能推荐**优先复盘的模块**（短板 + 超时模块）
- 提供「历史复盘库」：
  - 按试卷 / 模块筛选
  - 支持关键词搜索（基期、速算、转折、60 秒……）
  - 适合作为后期总复习的错题/动作库

### 5. ✅ 今日任务 & 打卡

- 每天自动生成「今日训练清单」，来源于「本周训练计划」
- 支持打勾打卡，系统根据完成情况：
  - 维护「连续打卡天数 streak」
  - 侧边栏显示 streak，增加一点点成就感
- 支持：
  - 自定义添加任务
  - 一键清空今日清单

### 6. 🗓️ 本周训练计划（自动）

- 系统基于最近 3 套卷，统计：
  - 各模块平均正确率
  - 各模块平均超时时间
- 自动挑出：
  - 短板模块
  - 时间黑洞模块
- 生成 7 天训练安排：
  - 每天固定三件事：资料速算 + 言语填空 + 当天重点模块专项
  - 每天都有「重点模块」标签，便于执行
- 支持一键导出周计划为 Markdown，方便拷贝到备忘录 / 便签

### 7. 📊 趋势分析

- 总分折线图：看整体是否在稳步上涨
- 各模块正确率折线图：观察单个模块是否还在崩盘/回暖
- 历史成绩明细表：日期、试卷名称、总分、用时、正确率，一目了然

### 8. ✏️ 录入成绩

- 按照真实考试结构录入：
  - 模块：政治理论、常识判断、言语理解（逻辑填空/片段阅读）、数量关系、判断推理（图推/定义/类比/逻辑）、资料分析
  - 每个模块：
    - 对题数
    - 实际用时
    - 计划用时（用于后续超时分析）
- 系统自动计算：
  - 总分（按固定权重）
  - 各模块正确率、总正确数、总题数、总用时等

### 9. ⚙️ 策略设置

- 可配置你的「考场规则」：
  - 数量：每题上限秒数
  - 资料：每篇上限分钟 + 是否超时先跳
  - 逻辑判断：每题上限秒数
  - 复盘统计天数：看板错因统计覆盖范围
- 所有自动建议（TOP3 卡片、明天训练、周计划）都会按你的策略来生成

### 10. 🛡️ 管理后台（可选）

- 仅管理员可见
- 支持：
  - 查看用户列表
  - 新增用户
  - 修改昵称 / 重置密码
  - 删除账号（保护 admin 不可删除）

## 🧱 技术栈 & 数据存储

- **后端框架**：[`Streamlit`](https://streamlit.io/)
- **数据处理**：`pandas`
- **可视化**：`plotly`（雷达图、折线图、直方图、饼图、条形图）
- **语言**：Python 3.x

数据存储方式（本地轻量级）：

- 用户信息：`users_db.json`
- 每个用户的成绩记录：`data_storage_<username>.csv`
- 每个用户的复盘记录：`review_notes_<username>.csv`
- 每个用户的策略配置：`strategy_<username>.json`
- 每个用户的打卡数据：`checkin_<username>.json`

> 不依赖数据库，拉下来本地运行即可使用，适合个人自用。
